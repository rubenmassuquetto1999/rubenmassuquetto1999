### 👨‍💻 Sobre mim

*A base de um bom código é como a fundação de uma casa: precisa ser sólida.* Sou um desenvolvedor focado em dominar a "tríade raiz" da web. Construo aplicações funcionais, com design responsivo e integração de APIs, sempre buscando resolver problemas reais através de código limpo. Meu objetivo é transformar lógica pura em experiências de usuário excepcionais.

- 🔭 Atualmente expandindo meu ecossistema **JavaScript** e aprimorando UI/UX.
- 🌱 Focado em estruturação e estilização impecáveis com **HTML, CSS e JS puro**.
