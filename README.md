### Forjando o Futuro 🌱

Sou um Desenvolvedor apaixonado por construir a web fundamental. Minha jornada é guiada pela criação de interfaces modernas, responsivas e funcionais, focando no que realmente importa: a base sólida. 

Este é o meu estúdio digital, onde transformo conceitos em código puro, forjando soluções rústicas e atraentes.

- 🔭 Atualmente criando aplicações web interativas e aprimorando habilidades de UI/UX.
- 🌱 Focado no ecossistema JavaScript puro, aprofundando meus conhecimentos em **HTML e CSS** para estrutura e estilo impecáveis.
- 📍 Baseado em Campo Largo, PR.
- 💬 Me pergunte sobre Front-end, JavaScript puro e design de interfaces!
